Tags HTML

As tags HTML (HyperText Markup Language) são usadas para estruturar o conteúdo de páginas web. Aqui estão as principais tags que você deve conhecer:

Lembrando: Os comandos devem ter "<" e ">" .

Índice

  • Tags de Estrutura Básica

  • Links e Imagens
  • Listas

  • Tags Semânticas HTML5

Tags de Estrutura Básica

(Voltar)
  • html = Define o início do documento HTML.
  • head = Contém metadados(título, links, scripts, etc.).
  • body = Contém o conteúdo visível da página.
  • title = Define o título que aparece na aba do navegador.

Tags de Texto

(Voltar)
  • h1 até h6 = Títulos de diferentes níveis (h1 é o mais importante).
  • p = Páragrafo.
  • br = Quebra de linha.
  • strong = Texto em negrito (semântico).
  • em = Texto em itálico (semântico).
  • span = Container genérico inline.
  • div = Container genérico em bloco.

Links e Imagens

(Voltar)
  • a = Cria um link (âncora).
  • img = Insere uma imagem.

Listas

(Voltar)
  • ul = Lista não ordenada (com marcadores).
  • ol = Lista ordenada (com números).
  • li = Item de lista.

Tabelas

(Voltar)
  • table = Define uma tabela.
  • tr = Linha de tabela.
  • td = Célula comum.
  • th = Cabeçalho da tabela.

Formulários

(Voltar)
  • form = Define um formulário.
  • input = Campo de entrada.
  • textarea = Campo de texto longo.
  • button = Botão.
  • label = Rótulo para campos.
  • select e option = Lista suspensa.

Tags Semânticas HTML5

(Voltar)
  • header = Cabeçalho da página ou seção.
  • nav = Navegação.
  • main = Conteúdo principal.
  • article = Conteúdo independete (ex: post).
  • section = Seção de conteúdo.
  • footer = Rodapé da página ou seção.
  • aside = Conteúdo lateral (ex: barra lateral).
